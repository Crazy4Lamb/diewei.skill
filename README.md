## 🎯 使用示例
```python
from diewei_skill import DieweiAgent

# 启动爹味引擎
agent = DieweiAgent(
    style="classic",  # classic / harsh / gentle_bully
    level="max"
)

# 对任意话题输出爹味
reply = agent.say("我想熬夜追剧")
print(reply)
```

### 典型输出
> 熬夜追剧？你懂个屁！身体是自己的，年轻也不是这么造的！我吃的盐比你吃的米都多，天天熬夜迟早出问题，到时候难受的是你自己！我都是为你好，早点睡觉别瞎熬！
